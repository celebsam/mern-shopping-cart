# Gadgetly

An e-commerce website for various gadgets including mobile phones and laptops.

## Appendix

This is a simple shopping website that allows users to add items to the cart and also place an order. Cart functionalities are only allowed for logged in users.

## Tech Stack

**Client:** React, Bootstrap, Redux

**Server:** Node Js, Express, MongoDB

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies for the backend

```bash
  npm install
```

Install dependencies for the frontend

```bash
  cd frontend
```

```bash
  npm install
```

Start the server

```bash
  cd ..
```

```bash
  npm run dev
```

## Features

- Mobile responsive
- User login
- Stores cart items in memory
